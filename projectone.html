MedFill Responsive Web Design

Creating a website to make prescription refills easier and more accessible.

↓


Overview

This website prototype was a project as part of Interaction Design Studio 1 at Carnegie Mellon University. My team was composed of four Bachelors and Masters of Human-Computer Interaction students. MedFill is a responsive web design that improves the current process of ordering refills from pharmacies.
